About Project

This project has been developed with the aim of compressing frontend code files, as removing spaces from frontend code does not pose a problem and they can be compressed. Therefore, this project works for files in the formats of HTML, CSS, and JS. HTML, CSS, SASS, and JS have been used in the frontend and Django has been used on the server side (backend) in this project.
